The files and folders you were just working on — including the ones in OneDrive, Dropbox, or any other cloud folder — always one click away.

Your latest files. One click away.

Get started

Up and running in under a minute

No configuration required — FanFolder works out of the box.

1

Run the installer

Run FanFolderSetup.exe — no admin rights needed. Done in seconds.

2

Click the taskbar icon

A FanFolder icon appears in your taskbar. Click it to see your recent files.

3

Customize via tray menu

Right-click the tray icon to change folder, sort order, animation style, and item count.

4

Starts automatically

FanFolder registers itself to start with Windows — nothing else to configure.
