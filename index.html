Features

A beautiful fan popup, right in your taskbar

Click the icon and your latest files and folders fan out instantly — documents, images, whatever you were last working on, including files from OneDrive, Dropbox, or any cloud folder.

Animated fan layout

Files arc out from the taskbar in a smooth fan, glide, spring or fade animation. Choose your style in settings.

Any folder

Point it at Downloads, Desktop, Documents, or your OneDrive, Dropbox, or any other cloud folder. Always shows your most recent files.

Full shell integration

Right-click any file for the complete Windows context menu. Drag files directly to other apps with a ghost preview.

Instant open

Icons are pre-loaded in the background so the fan opens the moment you click — no waiting.

Filter & sort

Sort by date modified, date created, or name. Filter to show only the files that matter to you.

Multilingual

The UI adapts automatically to your Windows display language — no extra setup needed.

Per-user install

Installs to your user profile — no admin rights needed. Starts automatically with Windows on every login.

Get started

Up and running in seconds

No configuration required — FanFolder works out of the box.

1

Download & run the installer

Run FanFolderSetup.exe — no admin rights needed. Done in seconds.

2

Click the taskbar icon

A FanFolder icon appears in your taskbar. Click it to open the fan and see your recent files.

3

Customize via the tray icon

Right-click the tray icon to change the folder, sort order, animation style, item count, and more.

4

Starts automatically on login

FanFolder registers itself to start with Windows — nothing else to configure.
