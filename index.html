Features

A beautiful fan popup, right in your taskbar

FanFolder runs silently in the background — lightweight, fast, and always ready.

🌀

Animated fan layout

Files arc out from the taskbar in a smooth fan, glide, spring or fade animation. Choose your style.

📁

Any folder

Point it at Downloads, Desktop, Documents, Recent Files, or any custom folder on your system.

🖱️

Full shell integration

Right-click any file for the complete Windows shell context menu. Drag files to other apps with ghost preview.

⚡

Instant open

Icons are pre-loaded in the background so the fan opens instantly with no delay.

🔍

Filter & sort

Sort by date modified, date created or name. Filter to show only what matters to you.

🌍

Multilingual

UI adapts automatically to your Windows display language.

🔒

Per-user install

Installs to your user profile — no admin rights required. Auto-starts with Windows on login.

Get started

Up and running in seconds

No configuration required — FanFolder works out of the box and auto-starts on login.

1

Download & run the installer

Run FanFolderSetup.exe — no admin rights needed. Installation takes a few seconds.

2

Click the taskbar icon

A FanFolder icon appears in your taskbar. Click it to open the fan and see your recent Downloads.

3

Customise via right-click

Right-click the tray icon to change folder, sort order, animation style, max items, and more.

4

Runs automatically on login

FanFolder registers itself to start with Windows — nothing else to configure.
